# Data Structures Implementation in C

A collection of fundamental Data Structures implemented from scratch in the C programming language.

## 📚 Overview

This repository contains practical implementations of commonly used data structures. The goal is to understand how each structure works internally without relying on external libraries.

## Implemented Data Structures

### Linked Lists

* Singly Linked List
* Circular Singly Linked List
* Doubly Linked List
* Circular Doubly Linked List

### Stacks

* Stack Using Array
* Stack Using Linked List
* Call Stack Simulation

### Queues

* Queue Using Array
* Queue Using Linked List
* Deque (Double Ended Queue)

### Trees

* Binary Search Tree (BST)

### Advanced Structures

* Trie
* Heap

## 🚀 Compilation

Compile any implementation using GCC:

```bash
gcc filename.c -o output
./output
```

Example:

```bash
gcc 01_Singly_Linked_List_Implementation.c -o linkedlist
./linkedlist
```
